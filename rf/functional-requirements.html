Functional Requirements for TAPWORK

User Registration​

  • The system shall allow users to register with their email address and a secure password.
  • Users must confirm their email address through a verification link sent to their inbox.

Barcode Generation​

  • The system shall generate unique barcodes for each registered user.
  • Barcodes must be scannable and linked to the user's profile in the database.

Email Notifications​

  • The system shall send email notifications for important events, such as registration confirmation, password resets, and attendance alerts.
  • Users should have the option to opt-in or opt-out of certain notifications.

User Management​

  • Administrators shall have the ability to manage user accounts, including creating, updating, and deleting user profiles.
  • The system shall provide a user-friendly interface for managing user roles and permissions.

Attendance Tracking​

  • The system shall track user attendance through barcode scanning.
  • Attendance records must be stored in the database and accessible for reporting.

Web Interface for Scanning​

  • The system shall provide a web interface that allows users to scan their barcodes using a webcam or mobile device.
  • The interface must be responsive and user-friendly.

Reporting​

  • The system shall generate reports on user attendance, including daily, weekly, and monthly summaries.
  • Reports must be exportable in common formats such as CSV and PDF.

Optional Biometric Features​

  • The system may include optional biometric features for user identification, such as fingerprint or facial recognition.
  • Biometric data must be securely stored and comply with relevant privacy regulations.
